
The following information indicates the progression of variable name manipulation to yield the final set of 
variable names below:

  1) The "()" characters were removed from the variable name and replaced with nothing (e.g. tBodyAcc-mean().X
     would become tBodyAcc-mean.X).
  2) The "-" characters were removed and replaced with a "." character (e.g. tBodyAcc-mean.X would become 
     tBodyAcc.mean.X).
  3) The abreviations were then converted to the full descrptive name (e.g. tBodyAcc.mean.X would become 
     Time Domain Body Accelleration.mean.X)

Note:  The number preceeding the variable name also refers to the column number that that variable can be found in.
